# 学習用なでしこパッド

## 基本的な使い方
- https://oza-jv.github.io/nakopad/ を開く。
- sampleフォルダ内のファイルをロードして起動するときは，https://oza-jv.github.io/nakopad/index.html?load=ファイル名.txt とする。
- リリース版は https://www.manabu-tech.net/nakopad/ で公開する。

## なでしこボード対応版
- 正式リリース（2021/7/29）
- plugin_nakoboard.jsが必要。
