// JMOOC講義用 WebAPIを利用して天気予報を取得する例
!非同期モード。

// 手順１ 画面の準備
都市一覧＝["札幌","東京","大阪","広島","福岡"]のセレクトボックス作成。
取得ボタン＝「の明日の天気は」のボタン作成。
結果ラベル＝「天気予報」のラベル作成。

// 手順2 WebAPIから天気予報を取得
API=「https://api.aoikujira.com/tenki/week.php?fmt=json」。
APIからAJAX受信。
天気＝対象をJSONデコード。

// 手順3 取得ボタンをクリックした時の動作
取得ボタンをクリックした時には
  
　　// 手順3-1 天気予報を文字で表示
　　選択都市＝都市一覧のテキスト取得。
　　明日天気＝天気[選択都市][0]["forecast"]。
　　結果ラベルに明日天気をテキスト設定。

　　// 手順3-2 天気予報を音声で話す
　　文章＝「{選択都市}の明日の天気は、{明日天気}」。
　　文章を話す。

　　// 手順3-2 予報によって色を変える
　　判定＝明日天気の１文字左部分。
　　もし、判定＝「晴」ならば
　　　　結果ラベルの「背景色」に黄色をDOMスタイル設定。
　　違えばもし、判定＝「雨」ならば
　　　　結果ラベルの「背景色」に水色をDOMスタイル設定。
　　違えば
　　　　結果ラベルの「背景色」に灰色をDOMスタイル設定。
　　ここまで。

ここまで。
