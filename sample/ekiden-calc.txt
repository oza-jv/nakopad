// 駅伝タイム計算機（簡易版）

// 各選手の記録を 時:分:秒 の形式で順に入力する
記録＝「00:11:00
00:11:32
00:10:56
00:10:17
00:11:57」。


// 順に加算して表示
計測＝記録をCSV取得。
累計＝「00:00:00」。
順番＝0。
計測を反復
　　順番＝順番＋1。
　　累計＝累計に「+{対象}」を時間加算。
　　「[{順番}] {対象} → 累計 {累計}」を表示。
ここまで。
改行作成。

// 累計を表示
「[累計] {累計}」を青色で表示。

// 平均を表示
平均＝(「00:00:00」と累計の秒差)/順番。
平均秒＝平均％60。
平均分＝(平均-平均秒)/60。
A＝「00:00:00」に「+{平均分}分」を日時加算。
A＝Aに「+{平均秒}秒」を日時加算。
「[平均] {A}」を赤色で表示。
